# AI Scientific News System

AI-driven automated scientific news generation and dissemination platform developed by THEK Research Institute.

---

# Overview

This project is an experimental scientific communication system designed to automatically:

- Search for relevant scientific news
- Generate AI-assisted scientific summaries
- Create visual publication assets
- Publish content automatically to Telegram channels

The platform integrates artificial intelligence, automated media systems, and scientific dissemination technologies into a unified workflow.

---

# Main Features

- Automated scientific news retrieval
- AI-generated scientific articles
- Telegram publishing automation
- Automated image integration
- Scientific media workflow
- Low-cost deployment architecture
